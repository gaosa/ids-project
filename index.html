The Ball Is Round - Really?

Prediction of soccer match results through data visualization and machine learning.

Goals

Sepp Herberger once said, “The ball is round, the game lasts 90 minutes. Everything else is pure theory.”

We aim to build a prediction system that can predict the match result given the historical match data.

About

Our dataset contains the matching data of England Premier League from the past seasons. The fields include information such as the name of teams, match results, match statistics (e.g. number of shots, number of corners, etc.) and handicap figures of large bet companies.

View details »

Summary

To further understand the dataset, we explored the data by visualizing some fields with Tableau. We analyzes the number of matches won by home team and away team, the total number of games won by each team participating in EPL, and the average goals per game for each team.

View details »

Analysis

We visualized the performance of different teams and the trends of change in playstyle. Specifically, we analyzed the win rate of home team and away team, the match statistics per team in each season, match results between different teams, fouls committed between different teams, and the odds given by different book makers.

Prediction

We performed data cleaning, feature engineering, data analysis, feature importance analysis of the dataset. Unfortunately, none of the strategies we implemented have achieved significant performance, which is not unsurprising.
